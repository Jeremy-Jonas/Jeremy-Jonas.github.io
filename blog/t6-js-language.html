JavaScript Language

Parentheses ()

Parentheses are used in conjunction with functions or loops and follow the syntax below.

  1. Syntax for execute a function: var result = doSomething()
  2. Syntax for a loop: for (loop conditions here)

Brackets []

Brackets are used to contain arrays, you can think of this like a shopping list, the "shopping list" is the name of the array, and the items on the list are what is contained within the array which is shown by the use of brackets. Eg: shoppingList:[milk, bread, apples]

Braces {}

The braces are used to enclose what a function or loop actually does. Eg: for (loop conditions here) {what actually happens while the conditions within () are met}

Single Quotes ''

Single quotes and double quotes actually do the same thing when surrounding a string (of text). It is essential that you use the same (single or double) at the beginning and end of the string however. An example of when to use single quotes for a string would be when you wash to include a quotation in your string - 'John said "Hello Mary!" and Mary waved.' As you can see if the string started with a double quotation then it would end at the space after the word said.

Double Quotes ""

An example of when to use double quotes is if you know that your string it to include an abbreviated word - "I don't understand how some people can't seem to use your and you're correctly!" As you see here if the string began with a single quotation then it would end in the middle of the word don't.
