What is Grid Based Design?

The best way to think of this is that there is an invisible grid on the webpage which the smaller boxes can align to. You can of course go deeper and imagine every box has an invisible grid of its own, and you can put as many boxes within a box as needed. This allows the developer to easily align and position elements on a webpage, and offers the added advantage that a total page layout overhaul can be done relatively quickly by simplay moving boxes around on the grid.

What's All The Hype About Responsive Webpages?

In this day and age it is essential that websites be designed with responsive pages. The number of people viewing webpages on mobile devices increases every day, so if you want a website that people will find useful, it needs to be able to transform depending on the size of the screen it is being viewed on. Responsive webpages are able to detect the size of the screen, and the type of device and change the entire layout, colour, text size and even hide certain content from the webpage to enable it to be viewed easily and navigated easily no matter the screen size.

What is Semantic Structure?

Semantic structure HTML uses elements which are easy to interperate by both the computer and human reading the code. This is achieved in a number of different ways, but the most common is by adding id="description of element" within the div tag of specific content on the webpage.
